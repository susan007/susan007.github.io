    插槽slot，是组件的一块HTML模块，父组件可在其插入html模板。

    # 单个插槽(匿名插槽)

    • 可放在组件任意位置
    • 一个组件中只能有一个该类插槽

    # 具名插槽

    • 有name属性
    • 一个组件中可出现多个，只要name的值不同

    # 匿名插槽和具名插槽实例

    父组件

    <template>
  <div class="father">
    <h3>这里是父组件</h3>
    <child>
      <div style="background-color: cornflowerblue">
        <span>菜单1</span>
        <span>菜单2</span>
        <span>菜单3</span>
      </div>
      <div slot="slot1" style="background-color: #44bb00">
        <span>乌拉拉</span>
        <span>乌拉拉</span>
        <span>乌拉拉</span>
      </div>
      <div slot="slot2" style="background-color: coral">
        <span>乌拉拉</span>
        <span>乌拉拉</span>
        <span>乌拉拉</span>
      </div>
    </child>
  </div>
</template>

<script>
  import child from './demo1-son'
  export default {
    name: 'demo1',
    components: {
      child
    }
  }
</script>

    子组件

    <template>
  <div class="child">
    <h3>这里是子组件</h3>
    <h4>匿名插槽</h4>
    <slot></slot>
    <h4>具名插槽</h4>
    <slot name="slot1"></slot>
    <slot name="slot2"></slot>
  </div>
</template>

<script>
  export default {
    name: 'child'
  }
</script>

    # 作用域插槽

    • 可在插槽上面绑定数据
    • slot-scope是自定义的 子组件
    <template>
  <div>
    <ul>
      <slot :items="items"></slot>
    </ul>
  </div>
</template>

<script>
  export default{
    name: 'child',
    props: {
      items: {
        type: Array
      }
    }
  }
</script>

    父组件

    <template>
  <div>
    <child :items="items">
      <!--之所以使用绑定是因为属性items的值是一个引用了Vue实例data里面的数组，而不是一个普通的字符串-->
      <template slot-scope="myProps">
        <li style="background-color: coral;border: 3px solid red" v-for="item in myProps.items">{{ item.text }}</li>
      </template>
    </child>
  </div>
</template>

<script>
  import child from './demo2-son';
  export default {
    name: 'father',
    data() {
      return {
        items: [
          {text: 'you are beautiful'},
          {text: 'yes, thanks'}
        ]
      }
    },
    components: {
      child
    }
  }
</script>
